Lab 6: Rolling Dice

September 24, 2021

Today we want to get a little practice writing functions with default values and exploring some of what the random library has to offer.

We’ll be resuming with some pair programming today, so check the board for your partner and location!

Repository Link

Part I: A Dice Roller

  • Write a function called roll_dice. It should take two arguments: the number of dice to roll and the number of sides all the dice will have.
  • The number of sides the dice will have should default to 6.
  • The function should return the sum of rolling all the necessary dice. For instance, if I rolled 2 6-sided dice and got a 3 and a 5, it should return 8.
  • It may help to add some print statements along the way to show what value is being rolled on each die as you go.
  • Make sure you add a descriptive doc-string (comment describing a function) explaining what the function does!

Part II: A Simple Game

  • Write a function called play_game which will have three parameters: the length of the game board, the number of dice allowed, the number of sides on each die (in that order).
  • The number of dice allowed should default to 2, and the number of sides on each die should again default to 6
  • Each time the set of dice are rolled, their total will move the player that far along the game “board”. Your function should keep rolling the sets of dice until the player reaches the end of the board, at which point it should return the number of turns it took to reach that point. You can assume that your player starts off the board at space 0, and the game finishes when they land on or past the board length.
  • Make sure you call your roll_dice function within this function. That’s why we wrote it above!
  • Before you return the number of turns taken, also print out a message saying how many turns were taken.
  • Make sure you add a doc-string to your function explaining what it does!
