Lab 8: Sentence Palindromes

October 8, 2021

A palindrome is a word or phrase that reads the same forwards and backwards. A classic example is the word "racecar", which has the same ordering of characters whether you read it forwards or backwards. These ideas can be extended to sentences or phrases if we ignore any punctuation, spaces, and differences in capitalization. "Madam, I'm Adam." is an oft cited example of a sentence palindrome, as stripping out everything but the letters leaves you with "madamimadam", which reads the same forward and backwards.

The goal for today is to write a predicate function which will return True if an input string is a palindrome (phrase or word) and False if not. You can find a template to get you started in the repository below.

Decomposition:

A problem like this has a quite natural decomposition. In order to achieve our goal, we need to be able to solve two, simpler problems: how can I check if a simple word is a palindrome? and how can I simplify a string to get rid of unwanted bits? So here we’ll approach them one at a time by writing helper functions, and then put it all together at the end.

Function 1

Write a function to check if a simple word or string (one with no spaces,punctuation, or capitalization) is a palindrome. Call this is_simple_palindrome. It should take one argument: the string to check, and return one boolean: True or False depending on whether the string is a palindrome or not. There are a variety of ways you can check to see if a string looks the same forwards and backwards: choose a method that makes the most sense to you. I’ve included a few tests in the if __name__ section at the bottom that you can check, but as an example, your output might look something like:

>>> is_simple_palindrome("racecar")
True
>>> is_simple_palindrome("windmill")
False

Function 2

Write function to simplify a sentence or phrase, by stripping out all symbols, whitespace, and capitalization. You can call this simplify_string() and it should take one argument: the string to simplify, and return the simplified string. Again, there are a few different ways you could approach this, but make sure that your final string contains purely lowercase letters. I’ve again included a few lines at the bottom of the program that you can uncomment to help check that this is working correctly. An example might look like:

>>> simplify_string("What a fine day this is!")
whatafinedaythisis
>>> simplify_string("  Oh @#$^&! you don't say!!")
ohyoudontsay

Putting it together

With those two helper functions working properly, it should be pretty simple to put them together into is_sentence_palindrome to achieve the desired goal!
